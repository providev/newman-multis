# Fun Multis

Juego interactivo para practicar las tablas de multiplicar. Construido con React + Vite + Tailwind CSS.

## Características

- Elige qué tablas quieres practicar
- 8 niveles con dificultad progresiva
- Modalidades: tipo test y escritura con teclado
- Límites de tiempo por nivel
- Personajes personalizables
- Seguimiento de progreso local

## Cómo empezar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
